# renke's generator-javascript

## Installation

```sh
npm install -g yo @renke/generator-javascript
```

## Usage

### Main generators

#### Node library

```sh
yo @renke/javascript:node-lib
```

#### Browser library

```sh
yo @renke/javascript:browser-lib
```

#### React library

```sh
yo @renke/javascript:react-lib
```

### Subgenerators

See [src](src) for all generators.
